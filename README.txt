1. Unzip the folder "producerconsumer".
2. Import the folder to the Eclipse as a "General Project" and add to the Work space.
3. Configure the Tomcat server to the eclipse 
 a. Go to windows-> Preferences
 b. Expand Server tab on the left and click on "Runtime Environment"
 c. Click on "Add" and Give the local path of the tomcat server
 d. click on "OK"
4. Right Click on the Project in the Prject Explorer
5. click on Run as-> Run on Server. 
6. Server should get started
7. Open the Localhost site on the browser(s).
